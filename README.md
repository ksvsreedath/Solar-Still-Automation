# Solar Still Automation System

## Overview
Solar still based desalination system with Arduino automation for water level management and brine drainage.

## Features
Solar powered operation
Autonomous control system
Low cost design (~₹3090)
Suitable for water scarce regions

## System Architecture
Solar Still → Evaporation
Glass Surface → Condensation
Sensors → Monitor water level
Arduino → Controls pumps

## Components
Arduino Nano
Water Level Sensor
Relay Module
Submersible Pumps
Solar Panel
Li-ion Battery

## Results
Water production: ~500 ml per day
Efficiency: ~21.48%
